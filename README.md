# Java Quiz Application

A simple console-based Quiz Application developed using **Core Java**.  
This project demonstrates menu-driven programming, arrays, methods, loops, switch-case, user input, and score calculation.

---

## Features

- Start Quiz
- Multiple Choice Questions (MCQ)
- Instant Correct/Wrong Feedback
- Displays Correct Answer
- View Final Score
- Percentage Calculation
- Input Validation
- Menu-Driven Interface

---

## Technologies Used

- Java
- Scanner Class
- Arrays
- Methods
- Loops
- Switch Case
- Conditional Statements (if-else)
